### About the project
![1](utils/1.jpg "Demo")
This java code make you able to run multiple commands on my shell such as:

* greet:
```bash
greet [Name]
``` 
it returns Hello , Name !

![2](utils/2.jpg "Demo")

* subtract:
```bash
subtract [number1] [number2]
``` 
it returns the substruction of the two numbers

![3](utils/3.jpg "Demo")

* multiply and divide:
```bash
multiply [number1] [number2]
divide [number1] [number2]
``` 
it returns the multiplication and division of the two numbers

![4](utils/4.jpg "Demo")

* External commands:
```bash
dir # for windows
ls # for linux
``` 
it returns the multiplication and division of the two numbers

![5](utils/5.jpg "Demo")

```bash
echo 
``` 

![6](utils/6.jpg "Demo")

in the end to quit you run 

```bash
exit 
```
